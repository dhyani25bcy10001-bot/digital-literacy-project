# TASK 3: CODING & COLLABORATION PLATFORMS 

For this task, I used HackerRank for coding practice and Google Forms for collaboration. On HackerRank, I completed a beginner-level problem, which helped me understand the platform and improve my basic coding skills. It also gave me confidence to attempt more problems in the future.

For the collaboration part, I created a Google Form titled “Digital Literacy Awareness Quiz” with five questions. The form included multiple-choice and short-answer questions to test students’ knowledge about digital safety and online behaviour. I also viewed the responses in Google Sheets, which showed how data can be collected and analysed easily.

These tools are very helpful academically. HackerRank improves logical thinking and coding skills, while Google Forms is useful for surveys, quizzes, and data collection in projects.

This task made me realize how digital tools can simplify learning and collaboration.
